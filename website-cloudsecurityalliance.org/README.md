# website-cloudsecurityalliance.org

Reference copies of security-related content on cloudsecurityalliance.org.

## How these are managed

- **`/.well-known/security.txt`** — Managed through Cloudflare's security.txt setting.
- **`/security/`** — Managed through CSA's internal CMS.

The files in this directory are reference copies for tracking changes, not the source of truth. Updates must be made through the respective platforms above.
